# FixMate AI - Desktop Installation Guide

## What is PWA (Progressive Web App)?

FixMate AI can be installed on your desktop like a native Windows application. Once installed, it will:
- Have its own desktop icon
- Open in a standalone window (no browser UI)
- Work offline with cached content
- Feel like a native desktop application

## Technical Details

- **App Name:** FixMate AI - PC Doctor
- **Type:** Progressive Web App (PWA)
- **Size:** ~15-20 MB (after caching)
- **Requirements:** Chrome 90+, Edge 90+, or other Chromium-based browsers
- **Offline Support:** Yes (with service worker caching)

## Troubleshooting

**Install button not appearing?**
- Make sure you're using HTTPS (required for PWA)
- Try refreshing the page
- Check if the app is already installed
- Clear browser cache and try again

**App not updating?**
- The app auto-updates when online
- You can manually refresh by closing and reopening the app
- Service worker will fetch new content in the background
